```

Data & Analytics: Electric Blue (#0080FF) → Charts, metrics
AI & Neural:     Neon Violet (#B026FF) → Intelligence indicators
Growth:          Success Green (#10B981) → Positive signals
Alerts:          Alert Orange (#FF006E) → Opportunities
```

### Typography Hierarchy
- Hero: 64px / 700 weight
- Heading: 48px / 600 weight
- Title: 24px / 600 weight
- Body: 16px / 400 weight / 1.6 line-height
- Label: 12px / 500 weight

### Premium Effects
- Border Radius: 14px (micro card luxury)
- Glass Morphism: 85% opacity + 10px blur
- Shadow System: 5 levels (micro to elevation)
- Motion: 250-300ms ease-out (purposeful)
- Hover Effects: -4px elevation + glow increase

### Visual Direction
- Bloomberg Terminal meets OpenAI dashboard aesthetic
- Neural network visualizations (AI focus)
- Cinematic gradients (deep indigo → obsidian)
- Soft glass effects (premium SaaS standard)
- Strategic motion depth (intelligent feel)
