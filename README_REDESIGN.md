✨ HOVER EFFECTS
═══════════════════════════════════════════════════════════════════════════════

Timeline: 300ms cubic-bezier(0.4, 0, 0.2, 1)

0ms:   User hovers
100ms: Scale begins (1 → 1.1x) + Overlay appears
200ms: Glow effect starts (opacity 0 → 1)
300ms: Complete - Scale 1.1x + Shadow-2xl + Full glow

Effects:
├─ Scale: 1 → 1.10x (10% larger)
├─ Shadow: shadow-sm → shadow-2xl
├─ Overlay: 0% → 15% opacity (gradient)
├─ Glow: 0% → 100% opacity (blur-xl)
└─ Cursor: Changes to pointer
